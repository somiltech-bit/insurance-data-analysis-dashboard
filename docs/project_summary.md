# Project Summary

Project Name: Insurance Data Analysis Dashboard

Dataset Size: 10,000 records

Tools Used:

- Excel
- Power Query Editor
- MS SQL Server
- Power BI

Objectives:

- Analyze insurance policy performance
- Study claim trends
- Compare policy types
- Evaluate claim statuses
- Analyze customer demographics
- Build an interactive dashboard

Business Outcome:

The dashboard helps identify high-performing insurance products, claim patterns and customer segments to support data-driven business decisions.
