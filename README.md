# Insurance Data Analysis Dashboard

## 📌 Project Overview

This project analyzes 10,000 insurance records to uncover insights related to policy performance, claim trends, premium collection and customer demographics.

The project follows an end-to-end data analytics workflow, starting from data cleaning and transformation to interactive dashboard creation for business decision-making.

---

## 🎯 Project Objectives

- Analyze insurance policy performance.
- Identify high-performing policy types.
- Study claim settlement patterns.
- Compare active and inactive policies.
- Analyze customer age groups with higher claims.
- Build an interactive dashboard for decision-making.

---

## 💡 Key Insights

- Travel insurance generated the highest premium revenue.
- Active policies exceeded inactive policies.
- Adults generated the highest claim amount.
- Rejected claims were higher than settled and pending claims.
- Travel and Health policies contributed significantly to total coverage.
- Young adults generated comparatively lower claims.
